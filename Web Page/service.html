Proyecto apto para todas las edades

Si estás buscando un proyecto para tus alumnos, ELCOche es el adecuado, apto para todas las edades por su flexibilidad en el desarrollo y capacidad de mejoras futuras.

Instrucciones de montaje

En nuestros productos incluimos el acceso a la guía de montaje para ELCOche para ayudar a los alumnos en el proceso de montaje.

Manual de usuario

Nuestros productos cuentan con una guía completa sobre cómo darle uso a ELCOche después del proceso de montaje.

Material para profesores

Además si eres profesor tendrás acceso al código completo para ayudar a tus alumnos en la programación.

Mantenimiento

Si encuentra algún problema con el funcionamiento del coche o piezas defectuosas, nuestro servicio técnico le ayudará, no dude en llamar.

Satisfacción asegurada

ELCOche es un proyecto de calidad y apto para todas las edades. Pruébalo! No te decepcionará.
